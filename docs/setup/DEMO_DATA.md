## 🧪 תרחישי בדיקה

### 1️⃣ בדיקת אפליקציית נהג
```
1. פתח: http://localhost:3010/driver.html
2. התחבר עם מספר טלפון: 050-1111111 / סיסמה: driver123
3. תראה את הנסיעה שלך (מחצבת נשר → רמת אביב)
4. לחץ על "יצאתי לטעינה" → סטטוס ישתנה ל-ENROUTE_PICKUP
5. המשך את התהליך: נטענתי → יצאתי לפריקה → השלמתי משלוח
```

### 2️⃣ בדיקת Web Admin
```
1. פתח: http://localhost:3010
2. התחבר עם: admin@fleet.com / admin123
3. עבור ל-Dispatch (לוח התכנון)
4. תראה את כל 5 הנסיעות להיום
5. נסה לערוך נסיעה או ליצור חדשה
```

### 3️⃣ בדיקת API
```powershell
# Login
$body = @{email="admin@fleet.com"; password="admin123"} | ConvertTo-Json
$res = Invoke-RestMethod -Uri "http://localhost:8001/api/auth/login" -Method POST -ContentType "application/json" -Body $body
$token = $res.access_token

# Get jobs
$headers = @{Authorization = "Bearer $token"}
Invoke-RestMethod -Uri "http://localhost:8001/api/jobs" -Headers $headers

# Get customers
Invoke-RestMethod -Uri "http://localhost:8001/api/customers" -Headers $headers
```

## 🔧 ניהול נתוני DEMO

### הרצה מחדש של נתוני DEMO
```bash
# אם רוצה לרוקן ולהתחיל מחדש
docker-compose down -v
docker-compose up -d
docker exec fleet_backend alembic upgrade head

# יצירת admin + נתוני demo
docker exec fleet_backend python /app/scripts/create_admin.py admin@fleet.com admin123
docker exec fleet_backend python /app/scripts/add_demo_data.py
```

### ניקוי רק נתוני DEMO (שמירת משתמשים)
```sql
# התחבר ל-DB
docker exec -it fleet_db psql -U fleet_user -d fleet_management

# מחק jobs
DELETE FROM jobs WHERE org_id = 1;

# מחק drivers (לא משתמשים)
DELETE FROM drivers WHERE org_id = 1;

# מחק משאיות
DELETE FROM trucks WHERE org_id = 1;

# מחק חומרים
DELETE FROM materials WHERE org_id = 1;

# מחק אתרים
DELETE FROM sites WHERE org_id = 1;

# מחק לקוחות
DELETE FROM customers WHERE org_id = 1;

# הרץ שוב
docker exec fleet_backend python /app/scripts/add_demo_data.py
```
